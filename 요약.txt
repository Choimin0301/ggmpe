ㄴ모양 - 렌더러들을 이용하여 보이게됨
ㄴ소리 - audio source 컴포넌트

=카메라 컴포넌트, 오디오 리스너 가짐

ㄴclear flags = 카메라에서 지울부분 선택

프로젝션이 2d일 경우 어떤 오브젝트가 더 앞에 그려질지만 영향을 줌, 3d는 원근 투영 적용

스크립트 설명
클래스 = 컴포넌트 단위
MonoBehaviour = 미리 유니티에서 만든 클래스
저장이 안될 시 스크립트 파일명 뒤에 *이 보임
저장 후 유니티로 돌아왔을 때 에러가 있으면 컴파일 후 콘솔창에 에러가 표시됨

초기화용 이벤트 함수
Awake(), Start(), OnEnble()
업데이트용 이벤트 함수
Update(), LateUpdate(), FixedUpdate()
오브젝트 파괴용
OnDestroy()
종료용
OnApplicastionQuit(), OnDisable()

Awake - 현재 씬에서 오브젝트가 활성화 되어있을 시 1회 사용, 데이터 초기화용
Start - 현재 씬에 오브젝트와 컴포넌트가 모두 활성화 되어있을 시 1회, 데이터 초기화용, 업데이트 지기전에 호출
OnEnable - 컴포넌트가 활성화 될 때마다 1회 호출
Awake -> OnEnble -> Start 순서로 호출

Update - 진행 도중 계속 실행되며 매 프레임마다 호출
LateUpdate - 업데이트 1회당 한번씩 호출됨
FixedUpdate - 툴바edit-Project Setting - Time에서 FIxed Timestep 변수로 반복 간격 설정 가능

OnDestroy - 파괴되어 사라질 때/씬이 변경되거나 게임이 종료될 때 호출됨

OnApplicationQuit - 게임 종료시 호출되는 함수
OnDisable - 게임 오브젝트/컴포넌트가 비활성화 될 때마다 호출됨
