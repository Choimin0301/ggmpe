Awake 함수 : 게임을 실행할 때 한 번 호출됨
주로 초기화용으로 사용
컴포넌트가 비활성화되어있어도 실행됨

Start 함수 : 오브젝트와 컴포넌트 모두 활성화되어있을 때 실행
주로 초기화용으로 사용
첫 번쨰 업데이트 함수가 실행되기 직전에 호출(Awake -> OnEnabe -> Start)

OnEnable 함수 : 컴포넌트가 비할성화 되었다가 활성화 될 때마다 1회 호출

Update 함수 : 컴포넌트가 활성화되어 있을 때 매 프레임마다 호출
(FPS 60 = 1초에 60번)

LateUpdate 함수 : 현재 씬에 존재하는 모든 오브젝트의 Update 함수가 1회 실행 된 후 실행
(활용 예 : 플레이어와 카메라같은 서로 다른 오브젝트가 존재할 때 플레이어를 쫓는 카메라를 구현 
플레이어가 Update로 이동할 때 카메라는 LateUpdate로 쫓아감)

FixedUpdate 함수 : 프레임의 영향을 받지 않음
Edit(툴바) - Project Settings - Time - Fixed Timestep 변수로 조절가능
(기본 값 0.02초, 0.02초에 1번 호출된다는 뜻)

OnDestroy 함수 : 오브젝트가 파괴될 때 1회 호출
씬이 변경되거나 게임이 종료될 때도 오브젝트가 파괴되기 때문에 호출됨

OnApplicationQuit 함수 : 게임이 종료될 때 1회 호출 
유니티 에디터에서는 플레이 모드를 중지할 때 호출

Ondisable 함수 : 컴포넌트가 활성화 되었다가 비활성화 될 때마다 1회 호출 (OnEnable 반대버전)
