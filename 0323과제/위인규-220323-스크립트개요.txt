스크립트 개요, 콘솔 뷰에 데이터 출력

Unity C# Script
-스크립트(Script)의 역할
  -스크립트가 컴포넌트로 종속된 게임 오브젝트에 주어지는 각종 명령 제어
  -게임 내에 사용되는 여러 오브젝트들을 생성, 삭제 및 관리
  -게임 전체 또는 일부를 관리하는 게임 내 시스템 구현

C# 스크립트 파일 생성
-Project View에서 Create 선택 C# 스크립트 선택
-파일 생성 직후 이름 변경시 스크립트 파일 내부의 클래스 이름이 파일 이름과 동일하게 변경
-파일 선택후 Visual Studio에서 스크립트 내용 편집 가능

C# 프로그래밍 언어 기초
- ; 을 붙여 문장을 마무리
-들여쓰기로 가독성 향상
-반드시 저장하기 스크립트 파일명 뒤에 *이 붙어있을 경우 ex) Test.cs* 저장 안됨을 뜻함 Ctrl + S 로 저장
-유니티로 돌아올때 자동 컴파일 후 에러가 있을 시  Console뷰에 에러 정보 출력
-// 또는 /* */로 주석 사용
-오브젝트에 스크립트 파일 생성 후 코드 Debug.Log("내용") 으로 콘솔뷰에 메시지 출력
-게임 오브젝트를 편집할 때는 플레이 종료 후 편집 플레이 중 편집 시 저장 내용 사라짐
